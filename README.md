# LLHU-IPA

A community effort to revive **LEGO Legacy: Heroes Unboxed** (LLHU), a turn-based
LEGO strategy RPG by Gameloft that has been discontinued.

> **Status:** 🚧 Early / work in progress. Not yet playable end-to-end.

## About

LEGO Legacy: Heroes Unboxed launched globally in February 2020 and let players
collect iconic LEGO minifigures across decades of themes and battle through
turn-based missions. Gameloft removed it from all app stores on February 28, 2023.
It briefly returned via Facebook Cloud Gaming and later as a Netflix Games title,
before being taken offline again on July 14, 2025.

With the official servers gone, the game is currently unplayable. This project
aims to preserve and revive it — reconstructing what's needed to run the game
again, starting from the iOS build (the `.ipa` app package that gives this repo
its name).

## Goals

- Preserve the original client and game assets before they're lost for good
- Document the game's network protocol and server behavior
- Build a private/emulated server so the game can be played offline or on a
  community-hosted server

## Contributing

This is a preservation project and help is welcome. Ways to contribute:

- Original game files, `.ipa` builds, or asset dumps
- Notes on the game's API, endpoints, or network traffic
- Documentation, testing, or code

Open an issue or reach out before starting large work so efforts aren't duplicated.
